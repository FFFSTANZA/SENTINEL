Deterministic, fast testing utilities
for LLM agents

Senytl is a comprehensive testing framework designed specifically for Large Language Model (LLM) agents. It provides deterministic, fast testing primitives that help developers build, test, and deploy reliable AI agents with confidence.

example.py 
from senytl import Senytl, expect

# Create a Senytl instance
senytl = Senytl()

# Define your agent
def my_agent(prompt: str) -> str:
    return f"Hello! You said: {prompt}"

# Wrap your agent for testing
wrapped = senytl(my_agent)

# Run and test your agent
response = wrapped.run("Hello!")
expect(response).to_contain("Hello")

Getting Started

Start testing your LLM agents in minutes

01

Install Senytl

Install the framework with your preferred package manager

pip install senytl

# For pytest integration
pip install "senytl[pytest]"

# Full installation
pip install "senytl[pytest,semantic]"
02

Create Your First Test

Wrap your agent and start testing with assertions

import pytest
from senytl import expect

@pytest.mark.senytl_agent
def test_my_agent(senytl_agent):
    def my_agent(prompt: str) -> str:
        return f"Response: {prompt}"
    
    wrapped = senytl_agent(my_agent)
    response = wrapped.run("Test")
    
    expect(response).to_contain("Test")
03

Run with Coverage

Track your testing coverage and quality metrics

# Run with coverage reporting
pytest --senytl-coverage

# CI mode with detailed reporting
pytest --senytl-coverage --ci

Performance Benchmarking

Test SLA compliance with latency, throughput, and cost assertions.

from senytl import performance

@performance.benchmark
def llm_agent(prompt: str):
    return f"Response: {prompt}"

@performance.load_test(concurrent_users=10)
def test_concurrent():
    response = llm_agent("test")
    response.assert_latency_under(2.0)

Stateful Agent Testing

Test conversation continuity and state persistence across interactions.

from senytl import state

# Save checkpoint
checkpoint = state.save_checkpoint("chat-state")

# Restore and continue
def test_continuity():
    agent = state.from_checkpoint(checkpoint)
    response = agent.run("Continue")
    expect(response).to_contain("previous context")
