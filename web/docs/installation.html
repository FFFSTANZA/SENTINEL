Installation

Get started with Senytl in minutes with pip install or your favorite package manager.

Requirements

  • Python 3.10 or higher
  • pip, poetry, or conda package manager
  • Optional: pytest for integration features

Basic Installation

Install Senytl using pip:

Terminal 
pip install senytl

Installation with Extras

Senytl offers several optional extras for enhanced functionality:

Pytest Integration

Recommended for seamless pytest integration:

pip install "senytl[pytest]"

Semantic Testing

For semantic similarity testing using embeddings:

pip install "senytl[semantic]"

Full Installation

Install all features:

pip install "senytl[pytest,semantic]"

Alternative Package Managers

Poetry

poetry add senytl

With extras:

poetry add "senytl[pytest,semantic]"

Pipenv

pipenv install senytl

Conda

conda install -c conda-forge senytl

Development Installation

For contributors and development work:

git clone https://github.com/senytl/senytl.git
cd senytl
pip install -e ".[pytest,semantic,dev]"

Verification

Verify your installation:

import senytl
print(f"Senytl version: {senytl.__version__}")

Dependencies

Senytl includes these core dependencies:

  • pyyaml - Configuration parsing
  • psutil - System and process utilities
  • tomli - TOML file parsing (Python < 3.11)

Optional extras:

  • pytest - Testing framework integration
  • torch - PyTorch for embeddings
  • sentence-transformers - Semantic similarity testing

Troubleshooting

Import Errors

If you encounter import errors, ensure all dependencies are installed:

pip install --upgrade senytl "senytl[pytest,semantic]"

Permission Issues

On Unix systems, you may need to use:

pip install --user senytl

Virtual Environment

Always use a virtual environment:

python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install senytl
