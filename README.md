# Senytl (Phase 1)

Senytl provides deterministic, fast testing primitives for LLM agents:

- A core mock engine for common LLM providers (OpenAI, Anthropic, Google) with fallbacks
- A pytest integration layer (plugin + fixtures + markers)
- An agent-centric assertion library
- Framework adapters (LangChain-style `invoke` plus raw Python agents)

This repository contains the minimal Phase 1 implementation.
